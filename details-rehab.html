Rehab Rescue

  • technology utilized: Javascript, React, Node.JS
  • year: 2023
  • repo: github

An interactive, customizable and user-friendly web app geared towards helping users overcome injury.

Birthed out of necessity after a kickboxing injury, i was sent to rehab and came home with a laundry list of stretches and exercises i needed to perform daily in order to heal. It was overwhelming! How would i ever remember to do them all, let alone do them every day?

Utilizing React and their recently added Hooks function, i developed an interactive, dynamic and easy-to-use web application that allows users to upload custom rehabilitation routines, add notes about specific exercises and gives the option to link a url image for easy visual recognition.

A checkbox input allows users to indicate what they have completed each day and a single click removes each exercise from the routine.
